API E HTTP

API é a junção das siglas application programming e Interface. É a forma de você interagir com servidores, aplicação, é a forma de comunicação de requisição.
HTTP é hyper text transfer Protocol, é um protocolo para enviar e receber dados na web. Temos alguns métodos:
GET: Pega a informação, método padrão e mais comum.
POST: Criar usuários e novos registros.
PUT: Atualizar informações, parecido com POST, mas esse só atualiza a informação.
DELETE: Deletar informações.
HEAD: Puxar o cabeçalho.
